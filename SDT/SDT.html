Musaic

Musaic allows users to create a meaningful art piece displaying their musical tastes through a personal mosaic.

For Musaic, I worked with my friend Ram as the UI Designer, UX Designer, and Front End Developer to create the website

Duration
Sept 2019 - Jan 2020
Role
UX Design, UI Design, Front End Development
Tools
Sketch, Invision, HTML, CSS, Javascript, Pug/Jade
Platform
Website
Team
2

Improvements

Most of these improvements are a result of Musaic being comprised of a two person team but nevertheless should be highlighted:

Using Different Streaming Services

To start developing Musaic, Ram and I had to decide which music platform to focus on. Since we were only a two person team, Ram said he would only be able to develop Musaic with one streaming service's API in order to roll out Musaic in a reasonable time.
